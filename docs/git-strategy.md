# Stratégie de branches

- Branche principale : `main` (production)
- Branche d'intégration : `dev`
- Branches features : `feature/*`

## Règles
- Interdiction de commit direct sur `main` et `dev`
- Pull Request obligatoire pour merger
- Nommage : feature/description-courte

## Flux
feature/* → dev → main
